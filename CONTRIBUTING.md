# Contributing

感谢你对 `agency-agents-zh-manage` 的关注与贡献。

本项目目前以 **实用、简洁、可维护** 为优先原则。提交变更前，建议先阅读以下说明。

## 贡献方向

欢迎以下类型的贡献：

- 修复脚本 bug
- 改进 README、示例与文档表达
- 提升错误提示与可用性
- 增强对 Codex / OpenClaw 的兼容性
- 改进 manifest 管理与角色安装体验

## 提交前建议

请尽量确保你的改动符合以下原则：

- 保持命令行为稳定，避免破坏已有用法
- 优先做小而明确的改动，避免一次性引入过多行为变化
- 文档与脚本改动保持一致
- 新增参数、命令或行为时，同时更新 README 或相关文档
- 尽量避免依赖个人本机目录结构或私有环境

## 提交流程建议

1. Fork 仓库并创建分支
2. 完成改动并自查
3. 确认相关文档已同步更新
4. 提交 Pull Request，并说明变更目的、影响范围与使用方式

## Pull Request 建议包含

- 变更背景：为什么要改
- 主要改动：改了什么
- 使用方式：命令或示例如何变化
- 兼容性说明：是否影响现有用户

## 文档风格

- 优先写清楚“这个功能解决什么问题”
- 示例尽量可复制、可直接理解
- 避免把仅维护者使用的发布检查项写进 README
- 中英文内容如同时维护，尽量保持结构一致

## Issue 建议

提交 issue 时，建议尽量附带：

- 使用场景
- 执行命令
- 实际结果
- 期望结果
- 运行环境（如 Codex / OpenClaw、Shell 环境等）
